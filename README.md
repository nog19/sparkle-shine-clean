# Thiago Estética &

Site institucional profissional para serviços de higienização de automóveis, estofados, colchões e tapetes. Desenvolvido com React, TypeScript, Tailwind CSS e shadcn/ui.

![Thiago Estética &](public/placeholder.svg)

## 🚀 Tecnologias

- **React 18** — Biblioteca para construção de interfaces
- **TypeScript** — Tipagem estática para JavaScript
- **Vite** — Build tool ultrarrápida
- **Tailwind CSS** — Framework de utilitários CSS
- **shadcn/ui** — Componentes UI reutilizáveis e acessíveis
- **React Router DOM** — Roteamento de páginas
- **TanStack Query** — Gerenciamento de estado server-side
- **Lucide React** — Ícones modernos e consistentes
- **Vitest** — Framework de testes unitários

## 📋 Pré-requisitos

- Node.js 18+ 
- npm ou bun

## 🔧 Instalação

```bash
# Clone o repositório
git clone https://github.com/ehopcdonog-jpg/sparkle-shine-clean.git

# Entre na pasta do projeto
cd sparkle-shine-clean

# Instale as dependências
npm install

# ou com bun
bun install
```

## 💻 Como executar

```bash
# Modo desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview da build
npm run preview
```

O servidor de desenvolvimento iniciará em `http://localhost:5173`

## 🧪 Testes

```bash
# Executar testes uma vez
npm run test

# Modo watch
npm run test:watch
```
